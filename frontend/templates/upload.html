Important: Excel Upload Guidelines

  • Sheet Names: Use Desktop, Server, and/or Switches as sheet names (case-sensitive).
  • Required Columns: Each sheet must have these columns:
    Desktop: Serial No., Company Name, Device Type
    Server: Serial No., Company Name
    Switches: Serial No., Company Name
  • Column Names: Must match exactly (e.g., Serial No., Company Name).
  • Duplicates: Duplicate Serial No. rows will be skipped automatically.
  • Missing Required Fields: Rows missing required fields will be ignored.
  • File Format: Only .xlsx or .xls files are accepted.
  • Tip: Download a sample template if unsure about the format.
